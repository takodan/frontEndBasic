# 0312 JS return; JavaScript 函式回傳值
使用函式基本須注意
1. 建立、呼叫
2. 參數
3. 回傳值
## return
```js
function functionName(p1, p2,...){
    // code block
    return; // return undefined
};
// as same as function without return

function functionName(p1, p2,...){
    // code block
    return data; // return data
};
```
